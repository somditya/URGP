- clean tables
- status badges

- visible focus states
- semantic HTML

- screen reader labels
- accessible forms

3. Improve spacing consistency
4. Improve form styling
5. Improve auth pages
6. Improve dashboard layouts
7. Improve responsiveness
8. Improve accessibility
9. Add professional UI polish
10. Ensure visual consistency

- do not use external UI libraries

- do not use Material UI

- Tailwind CSS only
- Lucide icons only
