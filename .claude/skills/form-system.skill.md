# Form System Skill

## Objective

Create reusable enterprise-grade form system for all workflows.

---

# Forms Required

- grievance registration
- appeal submission
- nodal officer resolution
- appellate authority decision
- profile management
- authentication forms

---

# Form Components Required

- Input
- Select
- Textarea
- Checkbox
- Radio Group
- Date Picker
- File Upload
- Multi File Upload
- Form Error
- Form Helper Text
- Form Section

---

# Upload Requirements

Allowed Files:

- pdf
- doc
- docx
- jpg
- jpeg
- png

Max File Size:
10 MB

# Validation Rules

- inline validation
- accessible error messages
- required indicators
- proper helper text
- password validation
- file validation

# Accessibility Requirements

- semantic forms
- keyboard navigation
- screen reader labels
- proper aria attributes
- visible focus states
- accessible validation messages

# UI Rules

- consistent field heights
- large touch targets
- responsive spacing
- section grouping
- clean visual hierarchy
- elegant enterprise appearance
