# Frontend Routing Skill

## Objective

Setup frontend routing architecture using React Router DOM.

---

# Required Routes

Public:

- /login
- /register
- /forgot-password

Protected:

- /dashboard
- /officer/dashboard
- /appeals

---

# Requirements

- modular route architecture
- protected route ready
- role-based route ready
- lazy-loading ready
- layout separation

---

# Tasks

1. Setup BrowserRouter
2. Create app router structure
3. Create route modules
4. Create placeholder pages
5. Create route constants
6. Create protected route wrapper foundation

---

# Do Not

- do not implement JWT yet
- do not implement API integration yet

Only routing architecture.
