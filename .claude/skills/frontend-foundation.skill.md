# Frontend Foundation Setup Skill

## Objective

Setup production-ready frontend foundation for the University Grievance Redressal Portal.

---

# Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router DOM
- Axios
- React Hook Form
- Lucide React

---

# Requirements

Create enterprise-grade frontend foundation.

Requirements:

- scalable architecture
- modular folder structure
- accessibility compliant
- responsive-ready
- reusable architecture
- maintainable codebase

---

# Tasks

1. Verify Vite React TypeScript setup
2. Install required dependencies
3. Configure Tailwind CSS
4. Setup global styles
5. Setup routing foundation
6. Setup layout architecture
7. Setup reusable UI component structure
8. Setup services architecture
9. Setup utilities and hooks structure
10. Ensure TypeScript strict compatibility

---

# Do Not

- do not create business logic yet
- do not create grievance workflows yet
- do not create APIs yet
- do not create dashboards yet

Only create frontend foundation.
