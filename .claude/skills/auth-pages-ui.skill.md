# Authentication Pages UI Skill

## Objective

Create Login, Register, and Forgot Password pages.

---

# Pages

- Login
- Register
- Forgot Password

---

# Requirements

- responsive
- accessible
- modern enterprise UI
- centered auth card
- reusable form structure

---

# Login Requirements

Fields:

- user type
- email
- password

Features:

- inline validation
- error states
- loading states
- forgot password link
- register link

---

# Register Requirements

Fields:

- user type
- full name
- email
- phone
- department/section
- password
- confirm password

---

# Forgot Password Requirements

Fields:

- email

---

# Styling

Primary color:
#1F4E79

Use:

- Tailwind only
- Lucide icons
- responsive design
- accessibility best practices
